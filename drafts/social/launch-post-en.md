# Launch Post Draft — English

I just open-sourced Dayan AI Workshop:

https://github.com/Kosmoray/dayan-ai-workshop

It is a small workshop for building AI agent workflows that can be replayed, checked, and trusted.

The core idea:

Context -> Action -> Control -> Verification -> Handoff

Not a prompt collection.
Not a private system dump.

It includes:

- a mission card for AI workers
- a permission redline library
- verifier templates
- a 30-minute quickstart
- a public-safe workflow map

If you are building AI coding workflows, internal AI employees, or multi-agent operations, this should save you a few painful loops.

Stars help more builders find it.
